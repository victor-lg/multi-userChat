<!-- INSTALLATION -->
## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/victor-lg/multiuserChat.git
   ```
2. Install NPM packages for both client and server
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm start
   ```
